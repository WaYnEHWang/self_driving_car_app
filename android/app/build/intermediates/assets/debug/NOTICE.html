宏碁自駕接駁車試乘注意事項

　　　　　　　　　　　　　　　　　　　　　　版本1.0 最後更新：2018年2月


請詳細閱讀下列試乘注意事項，當您按下『同意』，表示您已詳閱、瞭解並接受本注意事項。首次預約試乘後，後續預約視為已同意。若『不同意』，請勿預約試乘。

  1. 宏碁股份有限公司（下稱本公司）保有服務內容及方式之最終決定權。

  2. 為確保試乘品質，本公司現場工作人員及隨車人員有權安排試乘順序及座位。

  3. 本試乘服務僅限已開放試乘之對象預約試乘。

  4. 試乘者需遵守下列試乘規範：

  1. 需遵照現場工作人員及隨車人員之指示；

  2. 需透過「宏碁自駕接駁車試乘應用」APP預約試乘，並於APP預約之站點上下車；

  3. 若於預約時同意接受共乘，需接受共乘安排；

  4. 需全程繫上自駕接駁車安全帶；

  5. 未滿十二歲之兒童，需由成年人陪同搭乘；

  6. 行駛時試乘者之頭、身體及四肢與攜帶物品需保持於車體內。

  1. 試乘者若有下列情形，現場工作人員得拒絕或中止其試乘：

  1. 不遵守本注意事項之規定；

  2. 不遵照指示且經糾正仍不改善；

  3. 有騷擾其他試乘者或不當影響他人試乘體驗之行為；

  4. 攜帶之物品長度、體積影響試乘之安全或順利運作；

  5. 攜帶危險物品（如易燃物、具毒性、腐蝕性等物品）或依物品之性質有危害他人之身體或財產者；

  6. 攜帶腳踏車、滑板車或其他類似器材；

  7. 未經許可攜帶動物搭乘；

  8. 有其他違反法令、公共秩序或善良風俗之行為。　　　

  1. APP所提供的資訊，包括但不限於：預計抵達時間、行車動態、自駕車狀態等僅供參考，以實際試乘服務為準。

  2. 現場工作人員或隨車人員得依天候、路況或其他因素，隨時暫停、取消或中止試乘，已預約試乘者不得異議。

  3. 若試乘者因違反本注意事項之規定，造成人員傷亡或財產損失，需自行負擔法律責任，與本公司無涉。

  4. 若因不可抗力或其他不可歸責於本公司之原因造成試乘者之傷亡或財產損失，本公司不負任何法律責任。

  5. 本注意事項若有未盡事宜，依中華民國相關法規辦理。

  6. 本公司保留隨時修改本注意事項而不另行通知之權利。
